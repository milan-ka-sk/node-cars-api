# this Cars RESTful API has these endpoints:

# GET: --url--/cars
returns all cars

# GET: --url--/cars/--id--
returns a car with the defined ID

# POST: --url--/cars
adds one car 

# PUT: --url--/cars/--id--
updates a car with the defined ID

# DELETE: --url--/cars/--id--
deletes a car with the defined ID
